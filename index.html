A3FastSync

A3FS ist a open source mission and addon synchronization tool for ArmA 3

Features

  • A3FS uses HTTP for file transfer
  • resume partial downloads
  • (soon) limit bandwidth usage and parallel downloads
  • (soon) signed synchronization files (*.a3sync)
  • (soon) web interface
